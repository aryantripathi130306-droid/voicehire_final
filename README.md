VoiceHire is an AI-powered, voice-first employment platform designed to connect local workers (such as plumbers, maids, electricians, and vendors) with customers in their nearby area.

Unlike traditional platforms, VoiceHire focuses on accessibility for low-literacy users, allowing workers to create profiles and interact with the system using voice input instead of text.

Problem
Millions of skilled workers remain unemployed or underutilized because:

They cannot read or type in English

They lack digital presence

Existing platforms are not designed for them

Solution

VoiceHire enables workers to:

Create profiles using voice (speech-to-text)

Get discovered by nearby customers

Connect through direct calling (no chat needed)

Build trust via ratings and verification



Features
Voice-based profile creation

Location-based worker discovery

One-tap calling system

Simple and accessible UI

Rating & trust system



Tech Stack
Frontend: HTML, CSS, JavaScript / React

Backend: Firebase

Voice Processing: Web Speech API

Maps & Location: Google Maps API




Impact
VoiceHire aims to bridge the gap between digital platforms and the informal workforce, making job opportunities accessible to everyone—regardless of literacy level.




Future Scope
Multi-language voice support (Hindi, Punjabi, etc.)

Offline mode for low connectivity areas

AI-based job recommendations

Worker verification via government IDs
